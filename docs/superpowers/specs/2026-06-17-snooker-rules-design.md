# 斯诺克规则完善 — 设计文档

日期:2026-06-17

## 背景

斯诺克模式已部分实现(22 颗球、分值、红彩交替、彩球复位、清彩顺序、平分黑球复位、按分定胜负)。对照标准斯诺克规则,现有实现存在若干 bug 和缺口。本设计修正这些问题并补全可自动化的规则。

无法自动裁定的规则(无意识救球的"恢复重解"、僵局判定)本质需要裁判主观判断,2D 同屏无裁判,因此仅提供手动按键。

## 范围

| 部分 | 内容 | 主要文件 |

| 1 | 规则引擎修正 | `rules.py` |

## 第 1 部分:规则引擎修正(`rules.py`)

### 1a. 红球阶段打进彩球判犯规

现状(`rules.py:228-233`):红球阶段打进彩球会给彩球加分并复位。

修正:该打红球时(`phase == 'red'`),只要有 16–21 号球落袋 → 犯规。红球阶段唯一合法落袋是红球。

### 1b. 正确的犯规罚分

现状:`foul_points = max(4, snooker_value(first_contact))`。

修正:罚分取以下各项的最大值:
```
罚分 = max(4,
           value(目标球 ball-on),
           value(误碰的首球 wrong_first_contact),
           max(value(被误打进的各球)))
```

例:
- 该打红球,母球打丢只是停下 → 罚 4
- 该打红球,母球先碰黑球 → 罚 7
- 该打红球,误把粉球打进 → 罚 6

### 1c. 引入"目标球(ball-on)"概念

新增辅助函数 `snooker_balls_on(phase, next_color, balls)`,返回当前合法目标球号集合:
- 红球阶段:台面所有红球(1–15)
- 定彩阶段(`next_color` 非 None):那一颗指定彩球
- 自选彩球阶段(红球刚清空,`next_color is None`):台面所有彩球(16–21)

此函数同时供罚分计算(1b)与自由球检测(第 3 部分)使用。

## 第 2 部分:犯规后母球处理(`billiar_ball.py`)

现状(`billiar_ball.py:202-220`):每次犯规都进 `STATE_BREAK_PLACE` 让对手在 D 区重摆。

修正逻辑:

| 情况 | 母球位置 | 谁来打 | place_mode |
|------|---------|--------|-----------|
| 犯规但母球未落袋 | 停在原地不动 | 对手在原位接着打 | None |
| 犯规且母球落袋 | 回 D 区中心,对手在 D 区内摆放 | 对手 | 'kitchen' (+ STATE_BREAK_PLACE) |
| 正常未续杆 | 母球停在原地 | 对手 | None |

关键点:**只有母球自己落袋时**,对手才进 D 区摆球。其他犯规直接换人 + 进入 `STATE_AIMING`,母球留原地。斯诺克"原位接着打"时**不能重新拿起母球移动**,故 `place_mode = None`(区别于 8 球的自由球可任意摆)。

## 第 3 部分:自由球(Free Ball)

### 触发条件

上一杆对手犯规 **且** 轮到你时,台面对你形成"斯诺克障碍"(母球无法直线打到任一目标球两侧最薄边)。两条同时满足 → 获得自由球。

### 几何检测(`rules.py`,纯逻辑无 pygame)

```
is_snookered(cue, balls_on, all_balls):
    对每一颗 ball-on b:
        左切线、右切线 = 母球到 b 两侧最薄边的两条射线
        若两条射线中至少一条 不被其他球挡住 → 返回 False(没被障碍)
    所有 ball-on 两侧都被挡 → 返回 True(被障碍)
```

实现:母球中心到目标球中心连线,左右各偏移角度使射线擦过目标球边缘;用线段-圆相交判断该射线在到达目标球前是否先撞别的球(母球扫掠半径取 `2*BALL_RADIUS`)。

### 自由球玩法(`billiar_ball.py`)

- 新增标志 `self.free_ball = True`。
- 可把任意一颗球当作 ball-on 打,打进得**当前 ball-on 的分值**(红球还在时进彩球得 1 分;彩球阶段得那颗 ball-on 彩球的分值)。
- 被当替身打进的彩球**复位**(等同进了一颗红球)。
- 瞄准的"合法首碰"判定放宽:自由球阶段首碰任意球都合法。
- 提示文字:"自由球:可击打任意球作为目标球"。
- 自由球仅对触发后的那一杆有效,出杆后清除标志。

## 第 4 部分:`G` 键僵局重摆

`STATE_AIMING` 下按 `G`:本局作废,按原顺序重摆球、本局比分清零、回到开球摆球状态。直接调 `reset()`(reset 已保留跨局胜场 `self.scores`)。

"无意识救球"三选一中的"对手自己打"已由第 2 部分标准犯规流程自然支持(母球留原地接着打),故不单做 `F` 回滚。

## 第 5 部分:测试(`tests/test_rules.py`)

现无任何斯诺克测试。补充:
- 红球阶段进彩球 = 犯规
- 罚分取最大值(碰黑球罚 7、普通空杆罚 4、误进粉球罚 6)
- 红彩交替阶段推进
- 清彩顺序(黄→绿→棕→蓝→粉→黑)
- `is_snookered`:挡住 / 没挡住 / 部分挡住
- `snooker_balls_on` 在各阶段返回正确集合

## 不做(超出范围)

- 无意识救球的"恢复重解"/"让犯规方原位打"分支(需裁判判断)
- `F` 犯规前回滚
- 跳球、做斯诺克的力度判定等物理复杂度
